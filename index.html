Hamster, The Candy Hunter

→ : Go right
↓ : Go down
ENTER : Restart

Instructions :

  1. You start with 3 tokens.
  2. You only have 2 moves.
  3. Press the Right arrow key to go right and Down arrow key to go down.
  4. You have to hold the key and release where you want the hamster to stop.
  5. If the hamster lands on a candy you gain a token. You can win as many tokens.
  6. A token is deducted if the hamster does not land on a candy.
  7. Game ends when tokens reach 0.
